### Windows

```sh
1. cd my-wave-portal
2. npm install
3. type nul >> ".env"       # This will create a .env file
```

### Linux or MacOS

```sh
1. cd my-wave-portal
2. npm install
3. touch .env               # This will create a .env file
```

```sh
# This should be in your .ENV file
ALCHEMY_API=YOUR_ALCHEMY_API_KEY
PRIVATE_KEY=YOUR_WALLET_PRIVATE_KEY
```
